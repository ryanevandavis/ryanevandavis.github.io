CSS Position & Display Cheatsheet

Position Properties

Tag Name Description
position: static; Static Position Element is rendered in the normal flow of the document.
position: relative; Relative Position Element is positioned relative to its normal position in the document flow.
position: absolute; Absolute Position Element is positioned relative to the nearest positioned ancestor or the initial containing block.
position: fixed; Fixed Position Element is positioned relative to the initial containing block (usually the window) and remains fixed even when the page is scrolled.
position: sticky; Sticky Position Element is positioned based on the user's scroll position. Acts like relative: until a certain threshold is reached, then becomes fixed:.

Display Properties

Tag Name Description
display: block; Block Display Element takes up the full available width and starts on a new line.
display: inline; Inline Display Element is displayed inline, without line breaks.
display: inline-block; Inline-Block Display Element is displayed inline but behaves as a block-level element.
display: none; None Display Element is completely removed from the document flow and not visible.
display: flex; Flex Display Container becomes a flex container, allowing flexible layout and alignment of elements in one dimension.
display: grid; Grid Display Container becomes a grid container, allowing complex two-dimensional layouts with rows and columns.
display: table; Table Display Element behaves like a table element. Other elements can behave like table-row and table-cell elements.
display: inline-table; Inline-Table Display Element behaves like a table but inline. Other elements can behave like table-row and table-cell elements.
